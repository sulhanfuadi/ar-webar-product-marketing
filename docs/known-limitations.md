# Known Limitations (Single-Marker MVP)

- Full AR validation requires HTTPS and modern mobile browsers.
- Desktop remains preview-only and not a valid tracking benchmark.
- MVP supports one fixed marker/source image only (MacBook Air poster).
- 2D/3D panel in after-scan is placeholder content (not final product media stack).
- GLB model is optional; missing/invalid file falls back to runtime mesh.
- Chatbot flow is intentionally out of scope for this MVP iteration.
